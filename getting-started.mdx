## **FAQ**

<AccordionGroup>
  <Accordion title="Will the agent use the name I set in configuration?">
    Yes, the agent will use the name you configure in the agent settings. This name will be used when the agent introduces itself during conversations.
  </Accordion>

  <Accordion title="Does the voice need to match the language?">
    While it's recommended to match the voice language with the conversation language for better user experience, Mavera AI supports multilingual conversations with various voice options.
  </Accordion>

  <Accordion title="Why do advanced voice tweaks change quality?">
    Advanced voice tweaks modify the underlying voice synthesis parameters. Some adjustments may affect the naturalness or clarity of the voice. We recommend testing changes in simulations before deploying.
  </Accordion>

  <Accordion title="Why is my agent reaching voicemail?">
    This could be due to several reasons: the phone number might be busy, the call might be during non-business hours, or there might be connectivity issues. Check your telephony configuration and call logs for more details.
  </Accordion>

  <Accordion title="Why can't inbound agents use variables?">
    Inbound agents receive calls from unknown callers, so they don't have access to pre-populated variables. Variables are typically available in outbound campaigns where you have prior information about the contact.
  </Accordion>

  <Accordion title="Are there Mavera AI prompt templates?">
    Yes, Mavera AI provides various prompt templates for common use cases. You can find them in the agent configuration section.
  </Accordion>

  <Accordion title="Can I use my own telephony app?">
    Mavera AI supports integration with various telephony providers. Check our [Telephony documentation](/launch/telephony) for supported providers and integration guides.
  </Accordion>

  <Accordion title="Can I query subaccount minutes via API?">
    Yes, you can query subaccount usage and minutes through our API. See the [API Reference](/api-reference/getting-started-with-your-api) for details.
  </Accordion>

  <Accordion title="Do subaccounts get their own minutes?">
    Subaccounts can have their own minute allocations depending on your plan and configuration. Contact support@mavera.ai for details.
  </Accordion>

  <Accordion title="How are concurrent call limits shared?">
    Concurrent call limits are typically shared across your account or subaccount, depending on your plan. Enterprise plans may offer dedicated limits per subaccount.
  </Accordion>

  <Accordion title="When does billing occur each month?">
    Billing occurs on the same date each month based on when you first subscribed. Contact support@mavera.ai for billing inquiries.
  </Accordion>

  <Accordion title="How do I buy an Australian number?">
    Australian phone numbers can be purchased through the telephony section in your dashboard. Navigate to [Telephony](/launch/telephony) and select "Purchase Number" to see available regions.
  </Accordion>
</AccordionGroup>
