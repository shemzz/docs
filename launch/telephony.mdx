## **Overview**

Telephony configuration allows you to connect your Mavera AI agent to phone systems, purchase phone numbers, and manage call routing. This enables your agent to make and receive phone calls.

![Telephony Overview](/images/telephony-overview-placeholder.png)

## **Phone Numbers**

### **Purchasing Numbers**

1. Navigate to **Telephony** â†’ **Phone Numbers**
2. Click **"Purchase Number"**
3. Select country/region
4. Choose number type (local, toll-free, etc.)
5. Complete purchase

![Phone Number Purchase](/images/phone-number-purchase-placeholder.png)

### **Number Types**

- **Local Numbers**: Regional phone numbers
- **Toll-Free Numbers**: Free for callers
- **International Numbers**: Global coverage
- **Vanity Numbers**: Custom number patterns

### **Number Management**

- **Assign to Agent**: Link number to agent
- **Call Routing**: Configure routing rules
- **Voicemail**: Set up voicemail handling
- **Release**: Release unused numbers

## **Call Routing**

### **Routing Rules**

Configure how calls are handled:
- **Direct to Agent**: Route directly to agent
- **IVR Menu**: Interactive voice response
- **Time-based**: Different routing by time
- **Conditional**: Route based on conditions

![Call Routing](/images/call-routing-placeholder.png)

### **Voicemail**

Set up voicemail:
- **Enable Voicemail**: Allow voicemail recording
- **Voicemail Message**: Custom greeting
- **Notification**: Get notified of voicemails
- **Retrieval**: Access voicemail recordings

## **Call Settings**

### **Inbound Calls**

Configure incoming calls:
- **Answering Rules**: When to answer
- **Greeting**: Initial greeting message
- **Timeout**: Maximum wait time
- **Fallback**: What to do if agent unavailable

### **Outbound Calls**

Configure outgoing calls:
- **Calling Rules**: When to make calls
- **Caller ID**: Display number
- **Retry Logic**: Retry failed calls
- **Scheduling**: Schedule call times

## **Best Practices**
