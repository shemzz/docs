## **Overview**

Export call data to analyze conversations, generate reports, and integrate with other systems. Mavera AI provides multiple export formats and options.

![Export Overview](/images/export-overview-placeholder.png)

## **Export Options**

### **CSV Export**

Spreadsheet format:
- Easy to open in Excel/Sheets
- Good for basic analysis
- Includes conversation data
- Customizable fields

### **JSON Export**

Structured data:
- Machine-readable format
- Full conversation details
- Includes metadata
- API-friendly

### **PDF Export**

Document format:
- Human-readable reports
- Formatted transcripts
- Includes summaries
- Professional presentation

![Export Interface](/images/export-interface-placeholder.png)

## **Data Included**

- **Call Information**: Date, time, duration
- **Transcripts**: Full conversation text
- **Metadata**: Agent, phone numbers, etc.
- **Performance Data**: Metrics and analytics
- **Actions**: Actions executed during calls
