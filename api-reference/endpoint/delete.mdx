---
title: "Delete Agent"
description: "Delete an agent via API"
---

## **Endpoint**

<ParamField path="id" type="string" required>
  Agent ID
</ParamField>

<Warning>
  Deleting an agent is permanent and cannot be undone. All associated data will be removed.
</Warning>

<RequestExample>

```bash
curl -X DELETE https://api.mavera.ai/v1/agents/agent_123456 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

<ResponseExample>

```json
{
  "success": true,
  "message": "Agent deleted successfully"
}
```

</ResponseExample>
